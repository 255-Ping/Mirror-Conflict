## Core Concept

Each level spawns a mirrored opponent that exactly matches your inputs:

- Horizontal movement is inverted
- Vertical movement is mirrored
- Actions (jumping, interacting) are perfectly synced

Your copy is not AI-driven â€” it is **you**.

The challenge comes entirely from:
- Level layout
- Hazards
- Timing
- Spatial reasoning

- **Genre:** Puzzle / Action
- **Perspective:** 2D
- **Focus:** Using the environment against yourself
- **Win Condition:** Defeat or outmaneuver your mirrored self via level design
